#### Scalable landing template

* The current version uses only vanilla js web-components with client-side rendering and gets the component content from a separate json file, which url is specified in the src attribute. (Someday, it will be replaced with headless cms and ssr using astro or its analogue, which, undoubtedly, is the rightest approach for this case).

* Carousel component is fully optimized and uses only GPU-friendly opacity and transform transitions
